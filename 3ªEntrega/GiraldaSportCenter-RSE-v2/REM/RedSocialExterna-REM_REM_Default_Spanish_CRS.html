3 Descripción del Sistema Actual

El sistema actual, solo cuenta con un gestor de contenido, que parte de una web estática con informacón sobre el gimnasio, pero carece de funcionalidad sobre todo en la parte de interacción con el usuario y personal del gimnasio, por eso la necesidad de crear una red social que satisfaga esas necesidades.
